# Deep Learning Specialization Coursera [UPDATED Version 2021]
This repo contains all of my notebooks **the updated version** of Coursera's most famous [Deep Learning Specialization](https://www.coursera.org/specializations/deep-learning) of 5 courses offered by [deeplearning.ai](www.deeplearning.ai)
Instructor: [Prof. Andrew Ng](www.andrewng.org)

## Updated Assignment Notebooks
This Specialization was updated in April 2021 to include developments in deep learning and programming frameworks. One of the most prominient changes was shifting from Tensorflow 1 to Tensorflow 2.
Most of the old online repositories still don't have solved updated notebooks.
This repo contains solutions of update versions of the notebook (Relevant for 4th & 5th Courses).
